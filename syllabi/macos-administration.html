Syllabus: macOS Administration Fundamentals

IT Support Professional 24 hours DRAFT

Course Description

macOS Administration Fundamentals is a 24-hour hands-on course for IT professionals supporting Apple environments. Learners gain practical skills in installing and configuring macOS, managing user accounts, administering file systems and storage, implementing data backup strategies, managing applications and processes, configuring networking, and troubleshooting common issues. The course combines core concept instruction, hands-on labs, and group discussions to prepare technicians for real-world macOS support roles.

Summative Assessment: Capstone Lab and Final Assessment

The capstone lab presents a realistic scenario combining user account setup, network configuration, application management, and data backup. Learners demonstrate proficiency through hands-on troubleshooting of issues spanning multiple course topics. The final assessment includes a 10-question quiz, scenario-based capstone challenge, and comprehensive system configuration tasks. Passing requires demonstrated competency across all administrative domains.

Learning Outcomes

  1. Install, configure, and upgrade macOS using various deployment methods
  2. Create and manage local and network user accounts with appropriate permissions
  3. Manage APFS file systems, volumes, encryption, and file permissions
  4. Configure Spotlight, metadata, Time Machine backups, and archiving strategies
  5. Install, secure, and troubleshoot macOS applications and processes
  6. Configure and troubleshoot macOS networking (Wi-Fi, Ethernet, VPN, DNS)
  7. Manage peripherals, printers, startup modes, and command-line tools
  8. Apply macOS administration best practices and optimization techniques
  9. Troubleshoot common issues using built-in tools and diagnostics
  10. Prepare systems for enterprise deployment and management

Labs & Practical Exercises

  • macOS installation and configuration exercises
  • User account creation and permission management
  • File system and storage management exercises
  • Time Machine backup setup and testing
  • APFS encryption and FileVault configuration
  • Application installation and security configuration
  • Wi-Fi and Ethernet network configuration
  • VPN setup and network troubleshooting
  • Peripheral and printer configuration
  • Terminal command-line exercises
  • Activity Monitor and process management
  • Troubleshooting common macOS issues
  • System startup mode testing
  • Capstone lab: complete system setup and troubleshooting scenario
