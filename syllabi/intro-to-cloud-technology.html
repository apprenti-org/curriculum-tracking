Syllabus: Introduction to Cloud Technology

Cloud Operations Specialist DRAFT

Course Description

Introduction to Cloud Technology explores the core concepts and practical applications of cloud computing. Students will learn to differentiate between private and public clouds, understand the business advantages of cloud solutions, explore budgeting and cost estimation, examine cloud migration strategies, and master essential cloud models (IaaS, SaaS, PaaS, FaaS, CaaS). The course covers cloud servers, storage, containers, the shared responsibility model, and cloud certifications. This foundational course prepares learners for specialized cloud certifications and cloud operations roles across multiple platforms.

Summative Assessment: Cloud Technology Comprehensive Assessment

Students will complete a written assessment covering cloud concepts, cloud migration planning, service model comparisons, and cloud certification pathways. The assessment evaluates understanding of cloud strategy, service models, shared responsibility, and practical cloud computing applications. Students may also complete a practical component implementing a cloud solution or migration plan.

Learning Outcomes

  1. Differentiate between private, public, and hybrid cloud models
  2. Discuss the history and evolution of cloud computing
  3. Identify business benefits and ROI of cloud solutions
  4. Plan and estimate cloud budgeting and costs
  5. Execute and manage cloud migration projects
  6. Explain serverless computing models and use cases
  7. Contrast IaaS, SaaS, PaaS, FaaS, and CaaS service models
  8. Explain the shared responsibility model in cloud computing
  9. Identify benefits of cloud servers versus physical infrastructure
  10. Evaluate cloud storage, networking, and security options
  11. Explain containerization and container orchestration
  12. Identify and evaluate cloud certifications
  13. Stay current with cloud services and industry trends
