Syllabus: JavaScript React for C#

C# / .NET Full-Stack Development 40 hours DRAFT

Course Description

JavaScript React for C# is designed for C# developers transitioning to front-end development. This course covers JavaScript fundamentals, React component architecture, state management with hooks, and practical application development. The course draws parallels between C# and JavaScript concepts, helping .NET developers quickly become productive in front-end technologies. This course enables C# developers to build complete full-stack applications, bridging backend and frontend development.

Summative Assessment: Full-Stack React Application

Students will build a complete React single-page application that integrates with C# ASP.NET Core backend APIs. The assessment requires proficiency with React components, state management, API integration, and understanding of the C# to JavaScript ecosystem bridge. Learners must demonstrate understanding of HTTP communication between JavaScript frontend and .NET backend.

Learning Outcomes

  1. Understand JavaScript syntax and how it relates to C# concepts
  2. Work with JavaScript data types and type coercion
  3. Write functions and understand scope differently than C#
  4. Build React components and understand component composition
  5. Use React hooks (useState, useEffect) for state and lifecycle management
  6. Manage complex state in React applications
  7. Handle forms and user input in React
  8. Fetch data from ASP.NET Core APIs
  9. Implement routing for single-page applications
  10. Use React Context or Redux for state management
  11. Apply Material-UI or Tailwind for styling
  12. Debug JavaScript in browser tools
  13. Understand async/await and promises
  14. Build production-ready React applications
  15. Integrate React with C# ASP.NET Core backend services

Course Outline

1 JavaScript Fundamentals for C# Developers 4 hours
  • JavaScript vs C# syntax comparison
  • Variables and type coercion (vs strong typing in C#)
  • Functions (vs methods in C#)
  • Objects (vs classes in C#)
  • Arrays and collections
  • How JavaScript differs from C#
  • Type coercion and flexible typing
  • Function declarations vs C# methods
2 Control Flow and Functions 4 hours
  • Conditionals and loops
  • Function declarations and arrow functions
  • Closures and scope (different from C# scoping)
  • Higher-order functions and callbacks
3 Objects and Functional Programming 5 hours
  • Object literals (vs classes)
  • Prototypes (vs inheritance)
  • Array methods (map, filter, reduce)
  • Functional programming patterns
  • this keyword and binding
4 DOM and Events 4 hours
  • DOM manipulation (vs controls in WinForms/WPF)
  • Event handling
  • Event delegation
  • Working with HTML and CSS from JavaScript
5 Asynchronous JavaScript and APIs 5 hours
  • Callbacks, promises, and async/await
  • fetch() API (vs HttpClient in C#)
  • Consuming RESTful services
  • Integration with ASP.NET Core backends
  • Error handling in async operations
6 React Fundamentals 4 hours
  • React component architecture
  • Functional components
  • JSX (HTML-like syntax in JavaScript)
  • Props (passing data to components)
  • Component lifecycle concepts
7 React Hooks and State Management 5 hours
  • useState() hook (managing local component state)
  • useEffect() hook (handling side effects)
  • useReducer() for complex state (similar to C# event patterns)
  • Context API for global state (vs dependency injection)
  • Custom hooks
8 Building Complete Applications 8 hours
  • React Router for navigation
  • Form handling in React
  • API integration with React
  • Error handling and loading states
  • Testing React components
  • Styling with Material-UI or Tailwind
  • Deploying React applications
  • Build a complete CRUD application
  • Frontend in React with C# ASP.NET Core backend
  • Full request/response cycle

  • JavaScript Fundamentals: Practice syntax and patterns
  • DOM Manipulation: Create interactive pages
