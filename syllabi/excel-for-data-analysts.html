Syllabus: Data Fundamentals — Excel for Data Analysts

Data Analyst (DANL) 50 hours DRAFT

Course Description

Excel for Data Analysts teaches practical data analysis and manipulation skills using Microsoft Excel. Students will master data organization, Power Query for ETL workflows, pivot tables, charting, geographic analysis, statistical analysis, and correlation calculations. The course focuses on transforming raw data into actionable insights through cleaning, aggregation, and visualization. This course enables data professionals to leverage Excel's powerful features for business analysis and reporting across various industries.

Summative Assessment: Restaurant Ratings Analysis Project

Students will complete a comprehensive data analysis project analyzing restaurant ratings data. The assessment requires proper data organization, Power Query cleaning and transformation, pivot table creation, multi-chart visualization, statistical analysis, and correlation calculations. Learners must demonstrate proficiency in ETL workflows, ability to interpret analytical results, and skills in presenting findings through Excel charts and summaries.

Learning Outcomes

  1. Organize data in Excel following best practices
  2. Use Power Query for Extract-Transform-Load operations
  3. Differentiate between data cleaning and transformation
  4. Create and customize pivot tables for data summarization
  5. Build charts and visualizations in Excel
  6. Work with geographic data and maps
  7. Apply the Data Analysis ToolPak for statistical analysis
  8. Calculate correlation, rank, and percentile
  9. Create probability distributions and histograms
  10. Develop complete data analysis workflows in Excel
