Syllabus: Networking Fundamentals

CyberSecurity Analyst 220 hours DRAFT

Course Description

This Networking Fundamentals course provides a comprehensive introduction to core networking concepts and practical skills necessary for entry-level IT and networking roles. The 220-hour curriculum covers essential topics such as networking concepts, network implementation, network operations, network security, and troubleshooting.

Students will gain experience with routing and switching technologies, network monitoring, security protocols, and practical troubleshooting techniques. This course prepares students for real-world networking challenges and serves as a foundation for advanced certifications such as CompTIA Network+ and beyond.

Summative Assessment: Networking Fundamentals

The Networking Fundamentals Summative Assessment evaluates students' ability to apply core networking concepts through a practical exam (50%) and a written knowledge exam (50%). The practical component requires students to configure network devices, implement routing and switching protocols, troubleshoot connectivity issues, and apply security measures. The knowledge exam includes multiple-choice and scenario-based questions covering networking concepts, implementation, operations, security, and troubleshooting principles.

Students are assessed using a competency-based rubric that measures proficiency in key networking skills. A minimum passing score of 80% is required to demonstrate readiness for entry-level networking roles. This assessment ensures that learners can both perform technical networking tasks and apply theoretical concepts, preparing them for real-world networking environments and certification exams such as CompTIA Network+.

  1. Explain concepts related to the Open Systems Interconnection (OSI) reference model
  2. Compare and contrast networking appliances, applications, and functions
  3. Summarize cloud concepts and connectivity options
  4. Explain common networking ports, protocols, services, and traffic types
  5. Compare and contrast transmission media and transceivers
  6. Compare and contrast network topologies, architectures, and types
  7. Given a scenario, use appropriate IPv4 network addressing
  8. Summarize evolving use cases for modern network environments
  9. Explain characteristics of routing technologies
  10. Given a scenario, configure switching technologies and features
  11. Given a scenario, select and configure wireless devices and technologies
  12. Explain important factors of physical installations
  13. Explain the purpose of organizational processes and procedures
  14. Given a scenario, use network monitoring technologies
  15. Explain disaster recovery (DR) concepts
  16. Given a scenario, implement IPv4 and IPv6 network services
  17. Compare and contrast network access and management methods
  18. Explain the importance of basic network security concepts
  19. Summarize various types of attacks and their impact to the network
  20. Given a scenario, apply network security features, defense techniques, and solutions
  21. Explain the troubleshooting methodology
  22. Given a scenario, troubleshoot common cabling and physical interface issues
  23. Given a scenario, troubleshoot common issues with network services
  24. Given a scenario, troubleshoot common performance issues
  25. Given a scenario, use the appropriate tool or protocol to solve networking issues
