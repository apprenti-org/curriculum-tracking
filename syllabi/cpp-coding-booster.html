Syllabus: C++ Coding Booster Intensive

Software Developer 8 hours DRAFT

Course Description

The C++ Coding Booster Intensive is a one-day intensive workshop designed for engineers transitioning to or supporting C++ systems in enterprise environments, particularly transaction processing systems. This problem-based learning course uses realistic ServiceNow-style support ticket scenarios to teach practical C++ skills focused on data representation, memory management, and object-oriented programming. Learners gain fluency in C++ fundamentals while understanding how low-level concepts impact real-world systems at companies like Visa.

Summative Assessment: Final Challenge & Certification

Learners complete a comprehensive final quiz (15–20 questions) covering pointers, memory management, OOP, and practical workflows, along with a real-world support ticket challenge requiring investigation and fixing of logic errors in transaction processing system data formatting. Solutions are documented and submitted via GitHub pull request. Optional Certificate of Completion awarded for scores above 80%.

Learning Outcomes

  1. Explain the basic concepts of C++ and foundational coding principles
  2. Apply object-oriented programming (OOP) concepts to structure C++ applications
  3. Identify and use appropriate data types for different programming scenarios
  4. Describe how data is stored in memory, including frames, data sections, stack, and heap
  5. Store and manage data effectively within system memory
  6. Access and manipulate memory using pointers
  7. Convert data types and formats to meet system and performance requirements
  8. Translate transaction processing system (TPS) data into assembler-compatible logic
  9. Apply C++ concepts to build practical solutions to real-world problems
  10. Analyze business and operational requirements by working from ServiceNow-style support tickets before implementing technical solutions

Course Outline

1 C++ Coding Booster Intensive 8 hours
Lesson: Core Concept Overview (1.5 Hours)
  • Variables, data types, and memory allocation in C++
  • Data conversion techniques: type casting, formatting, binary representation
  • Pointers and memory addressing
  • Object-Oriented Programming: classes, objects, inheritance, encapsulation
  • Mini quiz on C++ data storage and manipulation fundamentals
Lesson: Hands-On Application (1.5 Hours)
  • Create a Transaction class with private attributes and public methods
  • Accept user input and convert data into binary-like structures
  • Use pointers to manipulate stored data
  • Real-world example: simplified TPS parser
  • Pair activity: Build a TransactionProcessor class
  • AI-assisted development and peer review
Lesson: Advanced Concepts & Comparisons (1.5 Hours)
  • Memory segmentation: code, data, stack, heap
  • Manual memory management (new/delete)
  • Assembler compatibility and data formatting
  • C++ vs Python and C++ vs Java/C# comparisons
  • Lab: Debug partially broken C++ code simulating a Visa support ticket
  • TDD with assert or lightweight GoogleTest
Lesson: Troubleshooting & Best Practices (1.5 Hours)
  • Common challenges: dangling pointers, memory leaks, type mismatches
  • Best practices: pointer initialization, nullptr, smart pointers
  • Document memory model and logic
  • Live troubleshooting: debug data corruption during casting
  • AI-assisted debugging and code comparison
  • Q&A on on-the-job application

Labs & Practical Exercises

  • Build a Transaction class demonstrating OOP and memory management
  • Create a TransactionProcessor class using pointers and type casting
  • Debug partially broken C++ code from real support ticket scenarios
  • Implement TDD with unit tests and assertions
  • Write final challenge solving a TPS data formatting problem
  • Peer review and AI validation of solutions
