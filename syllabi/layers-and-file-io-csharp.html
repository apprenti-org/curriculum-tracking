Syllabus: Layers and File I/O for C#

C# / .NET Fundamentals 40 hours DRAFT

Course Description

Layers and File I/O for C# teaches intermediate C# concepts essential for building professional applications. Students will master DateTime operations, collections (Lists, Dictionaries), application layering with class libraries, file I/O using System.IO, JSON and XML serialization, structs, and unit testing with NUnit. This course bridges foundational C# knowledge and advanced topics, emphasizing separation of concerns, testability, and data persistence. Students build on previous knowledge to create layered, maintainable applications.

Summative Assessment: Layered Application with Testing

Students will build a complete layered application demonstrating mastery of collections, file I/O, serialization, and unit testing. The assessment requires proper application architecture with separate UI, Business Logic, and Data Access layers; implementation of file persistence; appropriate use of collections; and comprehensive unit tests using NUnit. Learners must apply design patterns and demonstrate code organization principles.

Learning Outcomes

  1. Work with DateTime types and perform date/time calculations
  2. Use Lists and Dictionaries for data collection and storage
  3. Understand collection selection based on use cases
  4. Design layered application architecture (UI, BLL, DAL)
  5. Create and reference class libraries in .NET projects
  6. Read and write files using System.IO classes
  7. Implement JSON and XML serialization/deserialization
  8. Use structs appropriately for value-type data
  9. Write unit tests using NUnit framework
  10. Apply Test-Driven Development (TDD) principles
  11. Organize code following separation of concerns
