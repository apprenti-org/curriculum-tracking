Syllabus: JavaScript

Software Developer Java, Full-Stack Development 16 hours DRAFT

Course Description

JavaScript is a 16-hour foundational course introducing web scripting and client-side programming. Learners progress from JavaScript basics through DOM manipulation to JSON and object-oriented concepts. The course emphasizes hands-on code-alongs and exercises, with practical projects including a Dice Simulator and text-based games that reinforce core concepts of data types, functions, control flow, and DOM interaction.

Summative Assessment: Interactive Web Application Project

Learners complete hands-on exercises including a Dice Simulator, Coin Flip game, and Higher/Lower game demonstrating proficiency with functions, DOM manipulation, and event handling. Assessment evaluates ability to write functional JavaScript code, manipulate HTML elements dynamically, and apply control flow logic to interactive applications.

Learning Outcomes

  1. Understand JavaScript's role in web development
  2. Set up and use a JavaScript sandbox environment
  3. Work with string and number data types and operations
  4. Apply conditional statements and loops for program flow
  5. Create and manipulate arrays using common methods
  6. Define and call functions including arrow functions
  7. Refactor code using functions for reusability
  8. Select and modify HTML elements using DOM methods
  9. Add and remove DOM nodes dynamically
  10. Implement event listeners for user interaction
  11. Create JavaScript objects with key-value pairs
  12. Parse and stringify JSON data
  13. Apply JavaScript concepts to build interactive web applications

Course Outline

1 Introduction & Environment Setup 2 hours
Lesson: Course Introduction & JavaScript Sandbox Setup
  • Course overview and expectations
  • What is JavaScript and its importance for web development
  • Setting up JavaScript sandbox environment
  • Running JavaScript in browser console and VS Code
2 Strings and Numbers 2 hours
Lesson: Strings and Numbers
  • String declaration and common string methods
  • Number types and arithmetic operations
  • Type coercion and conversion
  • Template literals and string concatenation
3 Conditions and Loops 3 hours
Lesson: Conditions and Loops
  • Conditional statements: if, else if, else
  • Switch statements
  • Comparison and logical operators
  • for, while, and do-while loops
  • Loop control: break and continue
4 Arrays 3 hours
Lesson: Arrays & Dice Simulator
  • Array declaration and initialization
  • Accessing and modifying array elements
  • Common array methods: push, pop, shift, splice, slice
  • Code-Along: Building a Dice Simulator
5 Functions 2 hours
Lesson: Functions & Refactoring
  • Defining and calling functions
  • Parameters, arguments, and return values
  • Function expressions and arrow functions
  • Code-Along: Refactoring the Dice Simulator with functions
6 DOM Manipulation 2 hours
Lesson: Manipulating the DOM
  • Selecting HTML elements with getElementById and querySelector
  • Modifying element content, attributes, and styles
  • Adding and removing DOM nodes
  • Event listeners and user interaction
7 JSON, Objects & Practice 2 hours
Lesson: JSON and Objects
  • JavaScript object literals with key-value pairs
  • Accessing and modifying object properties
  • JSON.parse() and JSON.stringify()
  • Exercise: Coin Flip
  • Exercise: Higher/Lower

Labs & Practical Exercises

  • JavaScript sandbox setup and basic operations
  • String and number manipulation exercises
  • Conditional logic and loop practice
  • Array operations and the Dice Simulator project
  • Function definition and refactoring exercises
  • DOM selection and element manipulation
  • Adding event listeners for interactivity
  • JSON parsing and object creation
  • Coin Flip game implementation
  • Higher/Lower game implementation
  • Capstone: combining all concepts in interactive web applications
