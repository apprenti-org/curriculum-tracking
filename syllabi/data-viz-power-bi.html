Syllabus: Data Fundamentals — Data Visualizations and Power BI

Data Analyst (DANL) 50 hours DRAFT

Course Description

Data Visualizations and Power BI teaches learners to create effective data presentations using Power BI and design principles. Students will master stakeholder analysis, information design, dashboards, and the Power BI platform including slicers, groups, bins, and Excel integration. The course emphasizes communicating insights visually, working with large datasets, and publishing professional reports. This course builds on data literacy, SQL, and Excel foundations to enable learners to become proficient data visualization designers and Power BI developers.

Summative Assessment: Power BI Dashboard and Analysis Project

Students will create a comprehensive Power BI dashboard that demonstrates mastery of report design, visualization selection, interactive filtering, and data integration. The assessment requires effective use of slicers, groups, and bins to organize data; creation of multiple visualizations appropriate to the data; and clear communication of business insights through dashboard design. Students must demonstrate understanding of audience needs and application of design principles.

Learning Outcomes

  1. Create effective data visualizations for specific audiences
  2. Apply basic design principles to enhance data presentations
  3. Interpret infographics and analyze dashboard design
  4. Use Power BI to create and publish reports
  5. Implement slicers to filter and sort data interactively
  6. Use groups and bins to organize and aggregate data
  7. Integrate Excel workbooks with Power BI
  8. Work with large, complex datasets in Power BI
  9. Publish and share Power BI reports and dashboards
  10. Analyze census and real-world datasets using visualization
