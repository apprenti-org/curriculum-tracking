Syllabus: CompTIA Network+

IT Support Professional 96 hours DRAFT

Course Description

CompTIA Network+ is a comprehensive 96-hour course covering enterprise networking concepts, implementation, operations, security, and troubleshooting. Learners begin with foundational networking theory including the OSI model, protocols, and cloud architectures, then progress to hands-on routing, switching, wireless, and IP services configuration. The course emphasizes practical network operations, disaster recovery, access management, and systematic troubleshooting methodologies essential for IT professionals managing modern networks.

Summative Assessment: Network+ Certification Exam

Learners complete a comprehensive assessment combining practical labs with real-world scenarios and a certification exam simulation. The assessment evaluates mastery of all five modules: networking concepts, implementation, operations, security, and troubleshooting. Passing requires demonstrated proficiency across hands-on lab tasks, configuration exercises, and theory-based exam questions.

Learning Outcomes

  1. Explain the OSI model and TCP/IP model, including layer functions and encapsulation principles
  2. Identify and configure networking appliances including routers, switches, firewalls, and load balancers
  3. Design and implement cloud networking solutions including virtual networking and SDN
  4. Configure network protocols, ports, and services including IPv4, IPv6, DHCP, DNS, and NAT
  5. Apply transmission media standards and design network topologies
  6. Implement routing technologies and protocols for enterprise networks
  7. Configure switching technologies, VLANs, trunking, and redundancy mechanisms
  8. Deploy and troubleshoot wireless networking including Wi-Fi standards and encryption
  9. Implement network services including DHCP, DNS, NAT, and NTP
  10. Establish organizational processes and procedures for network change management
  11. Apply network monitoring tools and analyze performance using SNMP, syslog, and NetFlow
  12. Implement disaster recovery and business continuity strategies
  13. Configure and troubleshoot access management using AAA, RADIUS, TACACS+, and LDAP
  14. Identify and defend against network attacks and security threats
  15. Implement encryption, VPNs, and device hardening techniques
  16. Apply systematic troubleshooting methodology to diagnose and resolve network issues

Course Outline

1 Networking Concepts 20 hours
Lesson: OSI Model & Network Communication
  • Open Systems Interconnection model and protocol layers
  • TCP/IP model and comparison to OSI
  • Encapsulation and data flow between layers
Lesson: Networking Appliances & Applications
  • Routers, switches, firewalls, load balancers, and proxies
  • Network functions: NAT, VPN, IDS/IPS, QoS
Lesson: Cloud Concepts & Connectivity
  • Cloud models and virtual networking
  • Software-Defined Networking (SDN) and cloud service models
Lesson: Network Ports, Protocols, & Services
  • TCP/UDP ports and common protocols
  • Well-known and ephemeral ports
Lesson: Transmission Media & Network Topologies
  • Cabling standards and media converters
  • Star, bus, mesh, and hybrid architectures
Lesson: IPv4 Addressing & Modern Network Use Cases
  • Subnetting and CIDR notation
  • Emerging technologies (IoT, 5G, edge computing)
2 Network Implementation 20 hours
Lesson: Routing Technologies & Protocols
  • Static and dynamic routing
  • OSPF, EIGRP, BGP, and RIP
Lesson: Switching Technologies & Features
  • VLANs, trunking, and port mirroring
  • STP and redundancy
Lesson: Wireless Networking
  • Wi-Fi standards and encryption
  • Channel management and antenna types
Lesson: IPv4 & IPv6 Network Services
  • DHCP, DNS, NAT, NTP, and SNMP
  • IPv6 addressing and tunneling
3 Network Operations 20 hours
Lesson: Organizational Processes & Procedures
  • Change management and network documentation
  • Business continuity and SLAs
Lesson: Network Monitoring Tools & Technologies
  • SNMP, syslog, NetFlow, packet analyzers
  • Baseline and performance monitoring
Lesson: Disaster Recovery (DR) Concepts
  • Backups and failover strategies
  • Business impact analysis
Lesson: Network Access & Management
  • AAA and authentication protocols
  • Remote access methods
4 Network Security 20 hours
Lesson: Network Security Principles
  • CIA Triad and security policies
  • Risk assessments
Lesson: Types of Network Attacks
  • DoS/DDoS, phishing, MITM attacks
  • Social engineering and insider threats
Lesson: Network Defense Techniques
  • Firewalls, IDS/IPS, and anti-malware solutions
  • Secure network design
Lesson: Securing Network Traffic & Devices
  • Encryption and VPN technologies
  • Device hardening practices
5 Network Troubleshooting 12 hours
Lesson: Troubleshooting Methodology
  • Identify, diagnose, test, resolve, verify process
  • Documentation and escalation
Lesson: Common Cabling & Physical Issues
  • Cable testing and TDR tools
  • Diagnosing signal loss and interference
Lesson: Troubleshooting Network Services
  • DNS and DHCP issues
  • IP conflicts and misconfigurations
Lesson: Performance Issues & Tools
  • Packet loss and latency analysis
  • Using ping, traceroute, and Wireshark

Labs & Practical Exercises

  • Hands-on configuration of routers and switches
  • Implementation of VLANs and STP
  • Wireless network setup and security configuration
  • Network troubleshooting scenarios using diagnostic tools
  • Disaster recovery simulation exercises
  • Full-length practice exams and real-world scenario labs
