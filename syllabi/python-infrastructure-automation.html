Syllabus: Python for Infrastructure Automation

Operations Support Specialist — Network 32 hours DRAFT

Course Description

Python for Infrastructure Automation teaches IT operations professionals how to use Python to automate routine operational tasks, manage cloud infrastructure, and improve system reliability. This course builds on Python fundamentals and applies them to real-world infrastructure challenges including configuration management, monitoring, backup automation, and deployment processes. Learners will develop practical skills in working with APIs, cloud provider SDKs, and automation frameworks essential for modern operations teams.

Summative Assessment: Infrastructure Automation Project

Learners will design and implement a comprehensive automation solution for a realistic operational challenge. The project may involve automating provisioning, implementing monitoring and alerting, creating deployment automation, or developing cloud infrastructure management tools. The assessment evaluates Python proficiency, understanding of operations workflows, API integration skills, error handling, and code quality.

Learning Outcomes

  1. Apply Python fundamentals to systems administration tasks
  2. Develop scripts for file operations, text processing, and log analysis
  3. Implement network programming solutions for operational needs
  4. Automate configuration management and system provisioning
  5. Create monitoring, alerting, and health check automation
  6. Implement backup and disaster recovery automation
  7. Use cloud provider SDKs to manage infrastructure programmatically
  8. Develop infrastructure automation using Pulumi and other IaC frameworks
  9. Integrate Python automation into CI/CD pipelines
  10. Implement secure, resilient, and performant automation solutions
