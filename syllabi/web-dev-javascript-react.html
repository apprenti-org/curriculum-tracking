Syllabus: Web Development with JavaScript and React

Front-End / Full-Stack Development 40 hours DRAFT

Course Description

Web Development with JavaScript and React teaches modern front-end development from JavaScript fundamentals through building complete single-page applications. Students will master JavaScript syntax, control flow, functions and scope, object-oriented concepts, DOM manipulation, event handling, asynchronous JavaScript, and React component architecture. The course emphasizes practical application through progressive projects culminating in a capstone single-page application. This comprehensive course prepares learners for front-end and full-stack development roles.

Summative Assessment: Single-Page Application Capstone

Students will build a complete single-page application (Weather Dashboard or Product Browser) integrating JavaScript fundamentals, DOM manipulation, React components, API integration, and state management. The capstone requires functional components with hooks, data fetching and async handling, conditional rendering, and clean CSS styling. Learners must also participate in peer code review and presentation.

Learning Outcomes

  1. Write JavaScript code using proper syntax and conventions
  2. Use variables effectively with let, const, and var
  3. Work with primitive and complex data types
  4. Use conditional statements to control program flow
  5. Implement loops for iteration
  6. Define and call functions with parameters and return values
  7. Understand scope and closure
  8. Work with arrays and objects
  9. Use arrow functions and functional programming approaches
  10. Handle events in the DOM
  11. Manipulate the DOM dynamically
  12. Understand asynchronous JavaScript and promises
  13. Fetch data from APIs
  14. Build React components using hooks
  15. Manage state with useState and useEffect
  16. Debug JavaScript using browser tools
  17. Write basic tests for JavaScript code
  18. Build complete single-page applications

Course Outline

1 JavaScript Fundamentals 4 hours
Lesson: 1: Getting Started with JavaScript
  • Introduction to JavaScript
  • REPL and browser console
  • Basic syntax and writing scripts
  • Console logging
Lesson: 2: Variables and Data Types
  • var, let, const (declaration and scoping)
  • Primitive types: string, number, boolean, null, undefined
  • Type coercion and conversion
  • Dynamic typing
Lesson: 3: Data Structures — Arrays and Objects Intro
  • Arrays: creation, access, modification
  • Objects: key-value pairs, notation
  • Nesting and data structure design
2 Control Flow in JavaScript 4 hours
Lesson: 4: Making Decisions with Conditionals
  • if, else if, else statements
  • Comparison operators
  • Logical operators (&&, ||, !)
  • Nested conditions
Lesson: 5: Repeating Actions with Loops
  • for, while, do-while loops
  • foreach for iterating collections
  • Loop control (break, continue)
3 Functions and Scope 5 hours
Lesson: 6: Writing and Using Functions
  • Function declarations vs expressions
  • Arrow functions
  • Parameters and arguments
  • Return values
Lesson: 7: Function Scope, Hoisting, and Best Practices
  • Local vs global scope
  • Function scope and block scope
  • Hoisting behavior
  • Naming and organization best practices
4 Objects and Object-Oriented Programming 5 hours
Lesson: 8: JavaScript Objects and Their Properties
  • Object literals
  • Accessing/modifying properties
  • Dot vs bracket notation
  • Adding/deleting properties
Lesson: 9: Introduction to Object-Oriented Programming
  • Constructors and new keyword
  • Methods on objects
  • this keyword and context
  • Prototypes
5 Event Handling in JavaScript 4 hours
Lesson: 10: Understanding and Using Events
  • What is an event?
  • Common events (click, submit, change, mouseover)
  • addEventListener() and handlers
Lesson: 11: Event Types, Flow, and Delegation
  • Event propagation: bubbling vs capturing
  • event.target
  • preventDefault()
  • Event delegation
6 The JavaScript DOM vs HTML DOM 5 hours
Lesson: 12: Understanding the DOM and Traversing It
  • DOM structure and tree
  • Selecting elements (getElementById, querySelector)
  • Traversing nodes (parent, children, siblings)
Lesson: 13: Modifying the DOM Dynamically
  • Changing content and styles
  • Creating and appending elements
  • Working with attributes and classes
7 Testing and Debugging JavaScript 5 hours
Lesson: 14: Debugging JavaScript in the Browser
  • Error types (syntax, reference, type)
  • console.log() and typeof
  • Browser DevTools
  • Breakpoints and watch expressions
Lesson: 15: Introduction to Testing JavaScript
  • Manual vs automated testing
  • Simple test functions
  • Testing frameworks (Jest overview)
  • Assertions and test cases
8 JavaScript with React & APIs 8 hours
Lesson: 16: React Fundamentals and JavaScript Integration
  • React and component architecture
  • JSX syntax
  • Functional components
  • Props and state basics
  • JavaScript in JSX
  • Lists and user input
Lesson: 17: Fetching Data and Managing State
  • useEffect() for lifecycle
  • fetch() API
  • Asynchronous behavior
  • Conditional rendering
  • State management for APIs
  • Build a mini single-page application
  • Weather Dashboard or Product Browser
  • Group showcase and peer feedback
