Syllabus: C# Data Access

C# / .NET Fundamentals DRAFT

Course Description

C# Data Access teaches learners how to connect C# applications to relational databases using both low-level and high-level data access patterns. Students will learn ADO.NET for direct database communication, application configuration management, Entity Framework Core as an Object-Relational Mapping (ORM) framework, database migrations, and stored procedure integration. This course provides essential skills for building data-driven applications in the .NET ecosystem, covering the complete data access layer from connection management to CRUD operations.

Summative Assessment: Complete Data Access Layer Implementation

Students will design and implement a comprehensive data access layer for a C# application. The assessment requires both ADO.NET and Entity Framework Core implementations, proper configuration of database connections, migration management, and integration of stored procedures. Learners must demonstrate proficiency in connection string management, parameterized queries, entity configuration, and safe data access patterns appropriate for different application contexts.

Learning Outcomes

  1. Set up and configure database connections for C# applications
  2. Use ADO.NET to execute SQL queries and manage result sets
  3. Implement parameterized queries to prevent SQL injection
  4. Perform CRUD operations using ADO.NET and SqlCommand
  5. Manage application configuration using appsettings.json and IConfiguration
  6. Understand Object-Relational Mapping (ORM) concepts
  7. Configure Entity Framework Core and define entity models
  8. Use LINQ to query databases through EF Core
  9. Create and manage database migrations
  10. Write and execute stored procedures from C# code
