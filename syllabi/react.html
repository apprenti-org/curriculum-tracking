Syllabus: React

Front-End / Full-Stack Development 40 hours DRAFT

Course Description

React teaches modern front-end development using Facebook's React library. Students will master component architecture, state and props management, React hooks (useEffect, useReducer), form handling, routing with React Router, UI frameworks like Material-UI, and testing practices. The course emphasizes reusable components, separation of concerns, and building complete single-page applications. Students will apply these concepts through progressive projects building from basic components to full-featured applications.

Summative Assessment: React Application Project

Students will build a complete React application demonstrating mastery of components, state management, routing, and testing. The assessment requires proper component structure, effective props and state usage, form handling, API integration, testing coverage, and professional UI using a component library. Learners must demonstrate understanding of React best practices and ability to build scalable applications.

Learning Outcomes

  1. Understand React first principles and component-based architecture
  2. Create and compose reusable functional components
  3. Use props to pass data between components
  4. Manage state with useState and useEffect hooks
  5. Use useReducer for complex state management
  6. Lift state up and implement context for shared state
  7. Handle forms with controlled and uncontrolled components
  8. Implement client-side routing with React Router
  9. Use Material-UI components and theming
  10. Build complete single-page applications
  11. Fetch data from APIs and manage async operations
  12. Write tests for React components
  13. Deploy React applications
