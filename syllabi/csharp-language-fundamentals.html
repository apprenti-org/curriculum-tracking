Syllabus: C# Language Fundamentals

C# / .NET Fundamentals DRAFT

Course Description

C# Language Fundamentals is an introductory course covering the essentials of the C# programming language and .NET platform. Students will establish a development environment, learn syntax fundamentals including variables, types, and console I/O, master control flow with conditionals and loops, understand functions and scope, work with arrays, and apply all concepts to a capstone project. This course provides the foundation for all subsequent C# development, preparing learners to build console applications and transition to object-oriented and web-based programming.

Summative Assessment: Capsule Hotel Management Application

Students will build a comprehensive console application that manages capsule hotel operations. The assessment requires the application of variables, types, strings, conditionals, loops, methods, and arrays to solve a real-world problem. Learners must demonstrate proper code organization using methods, effective user interaction through console I/O, and the ability to manipulate data structures to track guest information and room availability.

Learning Outcomes

  1. Set up a C# development environment and create .NET console applications
  2. Understand the relationship between C# language and the .NET runtime
  3. Declare and use variables with appropriate types
  4. Perform type conversions and understand type inference
  5. Read user input and write formatted output to the console
  6. Manipulate strings using properties, methods, and formatting techniques
  7. Use conditional statements (if/else, switch) to control program flow
  8. Implement loops (for, while, foreach) for iteration
  9. Define and call methods with parameters and return values
  10. Create and manipulate arrays for data collection
