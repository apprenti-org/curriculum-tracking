Syllabus: CI/CD Pipeline Concepts

Operations Support Specialist — Network 10 hours DRAFT

Course Description

CI/CD Pipeline Concepts introduces learners to the principles and practices of continuous integration and continuous deployment. This course covers version control strategies, automated build and testing processes, deployment methodologies, and popular CI/CD tools used in modern software development and operations environments. Learners will understand how CI/CD pipelines improve code quality, reduce deployment risk, and enable rapid feature delivery while maintaining system reliability.

Summative Assessment: Pipeline Design and Implementation

Learners will design and implement a functional CI/CD pipeline for a sample application or infrastructure project. The assessment evaluates understanding of pipeline architecture, ability to select and configure appropriate tools, knowledge of deployment strategies, and practical implementation skills. Learners will demonstrate their pipeline through automated testing, successful deployments, and documented workflows.

Learning Outcomes

  1. Explain the principles of continuous integration and continuous deployment
  2. Design version control workflows and branching strategies for team collaboration
  3. Configure automated build and testing processes
  4. Select appropriate CI/CD tools for specific use cases
  5. Implement deployment strategies that minimize risk and downtime
  6. Monitor and troubleshoot CI/CD pipelines
  7. Document and communicate pipeline workflows to team members

Course Outline
