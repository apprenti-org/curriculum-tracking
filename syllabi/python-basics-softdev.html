Syllabus: Python Basics (for Software Dev)

Software Developer 48 hours DRAFT

Course Description

Python Basics (for Software Dev) is a 48-hour comprehensive course designed for software development careers. Learners progress from environment setup through core Python concepts including variables, data structures, control flow, functions, exception handling, and testing. The course emphasizes practical application with hands-on exercises and mini-projects, culminating in integrated assessments that combine multiple concepts into functional Python applications.

Summative Assessment: Mini-Projects and Assessment Labs

Learners complete three summative assessments: a Study Log Tracker (primary assessment), Collections Tracker mini-project, and Student Grade Tracker mini-project. Each assessment integrates course concepts including collections, functions, file I/O, and control flow. Assessments evaluate ability to solve multi-step problems, apply Python best practices, and create functional applications from requirements.

Learning Outcomes

  1. Set up and configure the Python development environment
  2. Write and run Python scripts using VS Code and the Python REPL
  3. Work with variables and primitive data types: int, float, str, bool
  4. Perform type conversion and casting appropriately
  5. Apply user input functionality and validate input data
  6. Manipulate strings using indexing, slicing, and string methods
  7. Use f-strings for formatted string output
  8. Create and manipulate lists using common methods
  9. Apply list comprehensions for efficient data processing
  10. Understand and use tuples and sets appropriately
  11. Create and work with dictionaries and key-value pairs
  12. Apply nested data structures for complex problems
  13. Write conditional logic using if, elif, and else statements
  14. Implement loops using for and while constructs
  15. Use break, continue, and pass control structures
  16. Define functions with parameters, arguments, and return values
  17. Apply default and keyword arguments
  18. Understand scope and variable lifetime
  19. Handle exceptions with try/except blocks
  20. Raise exceptions appropriately
  21. Read and write text files
  22. Use context managers for file operations
  23. Apply unit testing with unittest framework
  24. Practice test-driven development
  25. Integrate multiple Python concepts into complete applications
