Syllabus: Introduction to HTML & CSS

Front-End Development, Full-Stack Development 40 hours DRAFT

Course Description

Introduction to HTML & CSS is a 40-hour course focused on building a strong foundation in web standards and styling. Learners construct well-structured HTML5 documents and apply CSS to create beautiful, responsive web pages. The course progresses from foundational concepts through hands-on code-alongs and exercises, culminating in a capstone project where learners design and develop a fully functional, responsive static website published to GitHub.

Summative Assessment: Static Website Project

Learners design and build a fully functional, responsive static web page using HTML5 and CSS. The project must be properly structured with semantic HTML elements, responsive styling using media queries, and published to a GitHub repository with a descriptive README file. The assessment evaluates HTML structure, CSS styling, responsive design implementation, and version control proficiency.

Learning Outcomes

  1. Create a well-structured HTML document using appropriate HTML5 elements
  2. Apply CSS to style a web page and implement responsive design techniques
  3. Implement design elements such as images, fonts, and interactivity using HTML and CSS
  4. Demonstrate version control skills by committing code and pushing to GitHub
  5. Document project details and instructions using a descriptive README.md file
  6. Organize a small web development project using Git and GitHub effectively
  7. Deliver a fully functional and responsive static web page that works across devices
