Syllabus: Pandas

Data Analyst (DANL) 50 hours DRAFT

Course Description

Pandas teaches the essential Python library for data manipulation and analysis. Students will master the data science lifecycle, use Pandas for data exploration and ETL operations, aggregate and group data, and make informed visualization choices. The course uses real-world datasets including stock market data to teach practical skills in data cleaning, transformation, and analysis. This course enables data professionals to leverage Python and Pandas for efficient data workflows.

Summative Assessment: Data Analysis Project with Pandas

Students will complete a comprehensive data analysis project applying the entire Pandas workflow. The assessment requires data loading and exploration, cleaning and transformation, aggregation and grouping operations, and visualization selection and implementation. Learners must demonstrate understanding of the data science lifecycle and ability to communicate insights through Pandas-based analysis.

Learning Outcomes

  1. Summarize the data science lifecycle
  2. Install and import Pandas effectively
  3. Create and manipulate DataFrames and Series
  4. Read data into DataFrames from various sources
  5. Explore data using size, shape, and data type operations
  6. Clean and transform data with Pandas
  7. Handle missing data appropriately
  8. Merge and join DataFrames
  9. Aggregate and group data using groupby operations
  10. Create visualizations with Pandas
  11. Make informed choices about visualization types
  12. Analyze real-world datasets using Pandas
