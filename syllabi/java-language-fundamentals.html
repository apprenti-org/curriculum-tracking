Syllabus: Java Language Fundamentals

Software Developer Java, Full-Stack Java Developer 56 hours DRAFT

Course Description

Java Language Fundamentals is a 56-hour course covering essential Java programming concepts from foundational syntax through object-oriented design patterns. Learners progress from setting up the Java development environment through hands-on projects including a shopping cart application and text adventure game. The course emphasizes practical application, testing methodologies, exception handling, and comprehensive capstone projects that integrate all concepts into real-world software solutions.

Summative Assessment: Storage Lockers Capstone Application

Learners complete a comprehensive capstone assessment demonstrating mastery of all course concepts. The Storage Lockers application requires building a complete object-oriented system that demonstrates proper use of classes, arrays, methods, control flow, exception handling, and debugging. Assessment evaluates code structure, functional correctness, and ability to write single-file Java programs.

Learning Outcomes

  1. Explain programming fundamentals and how Java code compiles and executes
  2. Set up and use the Java development environment (JDK, IntelliJ IDEA)
  3. Apply Java syntax including statements, blocks, and naming conventions
  4. Work with primitive data types, variables, and type conversion
  5. Use operators and understand operator precedence
  6. Define and use enumeration types in practical applications
  7. Understand object references and memory management
  8. Apply string manipulation and StringBuilder techniques
  9. Read user input via console and validate input data
  10. Implement conditional logic using if-else and switch statements
  11. Apply iteration using for, while, and do-while loops
  12. Create and manipulate arrays and perform common array operations
  13. Define methods, parameters, and return types
  14. Practice method overloading and understand scope
  15. Write and execute manual unit tests
  16. Debug Java programs using IDE tools and stack traces
  17. Understand and handle exceptions appropriately
  18. Integrate all concepts into comprehensive object-oriented applications
