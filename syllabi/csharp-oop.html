Syllabus: C# OOP

C# / .NET Fundamentals DRAFT

Course Description

C# OOP teaches core object-oriented programming concepts essential for professional software development. Students will learn to design and implement classes, understand inheritance and polymorphism, work with interfaces, manage package dependencies through NuGet, and use enums. The course emphasizes encapsulation, abstraction, and extensibility through practical projects including games and management systems. This course prepares learners to build sophisticated, maintainable applications using OOP principles in the .NET ecosystem.

Summative Assessment: Object-Oriented Application Project

Students will design and implement a complete object-oriented application demonstrating mastery of classes, inheritance, interfaces, and enums. The assessment requires proper encapsulation, implementation of polymorphism, appropriate use of abstract classes and interfaces, and integration of NuGet packages. Learners must demonstrate understanding of object-oriented design patterns and the ability to create extensible, well-structured code.

Learning Outcomes

  1. Design and implement classes with fields, methods, and properties
  2. Understand and apply encapsulation and access modifiers
  3. Create and use constructors for object initialization
  4. Implement inheritance hierarchies with base classes and derived classes
  5. Use polymorphism through method overriding and interfaces
  6. Differentiate between abstract classes and interfaces
  7. Apply the Single Responsibility Principle in class design
  8. Use NuGet to find, install, and manage package dependencies
  9. Define and use enums to represent fixed sets of values
  10. Build complete OOP applications integrating multiple concepts
