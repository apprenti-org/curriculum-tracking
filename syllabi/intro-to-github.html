Syllabus: Introduction to GitHub

Software Development (Java, C#, JavaScript tracks) 8 hours DRAFT

Course Description

Introduction to GitHub teaches learners the foundational tools and practices of version control and collaborative development using Git and GitHub. Students will master command-line Git operations from repository initialization to branch management, understand GitHub features for individual and team workflows, resolve merge conflicts, and manage large files. This course is essential for all software developers, providing skills for code collaboration, team development, and professional software practices.

Summative Assessment: GitHub Repository Management Project

Students will demonstrate proficiency in Git and GitHub by creating and managing a repository with the following requirements: initialization and commits, branch creation and management, pull requests and code review, merge conflict resolution, and team collaboration workflows. The assessment evaluates ability to use both command-line Git and GitHub's web interface effectively.

Learning Outcomes

  1. Identify key terms and concepts related to Git and GitHub
  2. Explain the importance of version control for individual and team development
  3. Demonstrate basic Git commands in the command line
  4. Install and configure Git and GitHub on a machine
  5. Initialize repositories and make commits
  6. Push code to remote repositories
  7. Explore and assess GitHub repositories and their structure
  8. Create, manage, and delete branches effectively
  9. Resolve merge conflicts using both GitHub and command-line tools
  10. Clone, pull, and push changes to remote repositories
  11. Use GitHub's Large File Storage (LFS) for managing large files
  12. Collaborate with teammates using pull requests and code review
