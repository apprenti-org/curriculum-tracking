Syllabus: JavaScript Coding Booster Intensive

Software Developer, Front-End Development 8 hours DRAFT

Course Description

The JavaScript Coding Booster is a modular one-day intensive focused on practical application of JavaScript skills for front-end scripting and web development. Learners replicate real-world Scrum workflows including feature-based development, test-driven development (TDD), and daily cloud deployments via GitHub. The curriculum targets JavaScript skills for environments like Microsoft Dynamics 365, with mandatory AI-assisted coding (85% of workflow) using GitHub Copilot and ChatGPT for design, development, and debugging.

Summative Assessment: Code Review & Cloud Deployment

Learners submit feature implementations with code review ensuring new code doesn't break existing functionality. Features are deployed to the cloud via GitHub pull requests and branch merging. Assessment evaluates code quality, input validation correctness, test coverage, Microsoft API familiarity, and ability to work effectively in Scrum teams.

Learning Outcomes

  1. Apply basic JavaScript skills for front-end scripting, focusing on input validation and design
  2. Operate effectively in a Scrum team, executing TDD and cloud deployment workflows
  3. Utilize Generative AI tools for prompting, ideation, validation, and code generation
  4. Navigate and contribute to established systems with acceptance criteria and live codebases
  5. Address technical needs related to Microsoft Dynamics 365 and CRM customization

Labs & Practical Exercises

  • Scrum team formation and ecosystem setup
  • AI-assisted design using ChatGPT for flowcharts
  • GitHub Copilot code generation for input validation
  • Write tests for JavaScript input validation logic
  • Implement front-end features in simulated Dynamics 365 environment
  • Code review and testing workflows
  • GitHub commit, branch management, and pull requests
  • Deploy features to cloud via GitHub
  • Microsoft API integration exercises
  • Team standup and retrospective participation
