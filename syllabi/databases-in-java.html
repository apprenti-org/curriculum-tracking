Syllabus: Databases in Java

Software Developer Java (JVFD) 40 hours DRAFT

Course Description

This course guides students through essential concepts and practical applications of working with databases in a Java environment. Spanning ten content modules, students explore topics ranging from database fundamentals and relational modeling to advanced SQL techniques and Java Database Connectivity (JDBC).

Starting with virtualization, containers, and Docker, the course progresses through database types and terminology, relational modeling with normalization and entity relationship diagrams, and hands-on installation and configuration of MySQL and SQL Server Express. Students gain deep proficiency in SQL — from basic SELECT statements through JOINs, aggregation, subqueries, and data manipulation (INSERT, UPDATE, DELETE). The course culminates with stored procedures, transactions, and connecting Java applications to databases using JDBC and Spring's JdbcTemplate.

Summative Assessment: Java Databases

Students complete a summative assessment implementing and testing database connectivity portions of a partially written restaurant management application (SimpleBistro). Students must implement provided database interfaces, create mappers for each table, and write unit tests for the implemented interfaces. The assessment evaluates the student's ability to apply JDBC, JdbcTemplate, and repository patterns to a real-world application scenario.

Learning Outcomes

  1. Explain the differences between relational and non-relational database types.
  2. Apply normalization rules through third normal form to design relational database schemas.
  3. Create and maintain databases using Data Definition Language (DDL) in both MySQL and SQL Server Express.
  4. Write SQL SELECT statements with filtering, sorting, and pattern matching.
  5. Combine data from multiple tables using INNER, LEFT, RIGHT, and CROSS JOINs.
  6. Use aggregate functions with GROUP BY and HAVING to summarize data.
  7. Perform data manipulation using INSERT, UPDATE, and DELETE statements.
  8. Write and call stored procedures with parameters.
  9. Apply SQL transactions with COMMIT and ROLLBACK for safe data modification.
  10. Connect a Java application to a database using JDBC and Spring JdbcTemplate.
