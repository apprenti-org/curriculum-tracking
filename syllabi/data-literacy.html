Syllabus: Data Fundamentals — Data Literacy

Data Analyst (DANL) 25 hours DRAFT

Course Description

Data Literacy is an introductory course that establishes foundational knowledge for anyone working with data. Students will learn the fundamental concepts of data, metadata, and information; explore database terminology and structure; understand how bias affects data analysis; examine various data gathering and sourcing techniques; study data relationships and modeling; and identify appropriate visualizations. This course serves as the prerequisite for all advanced data courses and is designed for learners with no prior data experience but basic computer skills.

Summative Assessment: Data Analysis Capstone Project

Students will complete a comprehensive capstone project called "Cooking by the Book" that integrates concepts from all seven modules. The summative assessment includes a 70-question quiz covering all course topics with a 120-minute time limit. The capstone requires learners to demonstrate understanding of data concepts, data sourcing, bias recognition, data relationships, and visualization selection in a real-world context.

Learning Outcomes

  1. Explain the concepts of data, metadata, and information
  2. Discuss database terminology and how databases organize information
  3. Discuss the importance of questioning data and recognizing its limitations
  4. Identify and evaluate data gathering and sourcing techniques
  5. Recognize and analyze bias in data and understand associated risks
  6. Define and identify data relationships
  7. Organize and model data in meaningful ways
  8. Choose appropriate visualizations for different data types and audiences
  9. Apply statistical thinking to data analysis
  10. Recognize trends, outliers, and abnormalities in data
