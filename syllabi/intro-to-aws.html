Syllabus: Introduction to AWS Cloud Platform

Cloud Operations Specialist, Data Analyst (DANL) DRAFT

Course Description

Introduction to AWS Cloud Platform provides comprehensive coverage of the Amazon Web Services ecosystem and core cloud services. Students will explore the shared responsibility model, configure and manage EC2 instances, design VPC networks with security groups and subnets, implement auto scaling and load balancing, utilize S3 for object storage, manage identity and access through IAM, work with SNS notifications and SQS message queues, create serverless functions with Lambda, and understand AWS pricing and governance. This course prepares learners for cloud operations roles and provides practical experience with enterprise cloud infrastructure.

Summative Assessment: AWS Cloud Infrastructure Project

Students will design and implement a complete AWS cloud solution that demonstrates mastery of multiple AWS services. The assessment requires proper EC2 configuration, VPC setup with security controls, S3 bucket configuration, IAM policy creation, integration of SNS/SQS for messaging, and Lambda function deployment. Learners must demonstrate understanding of the shared responsibility model, cost management, and cloud security best practices.

Learning Outcomes

  1. Explain cloud computing and the AWS shared responsibility model
  2. Navigate and access AWS services effectively
  3. Configure and manage EC2 instances and auto scaling groups
  4. Design VPC networks with appropriate security configurations
  5. Implement load balancing and high availability patterns
  6. Manage object storage using S3 buckets and lifecycle policies
  7. Configure IAM users, roles, and permissions
  8. Implement messaging patterns using SNS and SQS
  9. Create and deploy serverless functions with Lambda
  10. Understand AWS pricing models and cost management tools
  11. Apply governance and compliance controls in AWS
