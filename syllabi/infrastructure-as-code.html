Syllabus: Infrastructure as Code Fundamentals

Operations Support Specialist — Network 22 hours DRAFT

Course Description

Infrastructure as Code Fundamentals teaches learners how to manage and provision IT infrastructure through code rather than manual processes. This comprehensive course covers IaC principles, Terraform, Ansible, and cloud-native tools, enabling operations professionals to build scalable, repeatable, and version-controlled infrastructure. Learners will understand how to integrate IaC into CI/CD pipelines, implement testing and validation strategies, and manage multi-environment deployments across different cloud providers.

Summative Assessment: Multi-Environment Infrastructure Project

Learners will design and implement a complete infrastructure deployment using Terraform and Ansible across multiple environments. The assessment evaluates understanding of IaC concepts, proficiency with major tools, ability to design modular and maintainable infrastructure, and integration with CI/CD practices. Learners will document their infrastructure, demonstrate deployments, and explain design decisions.

Learning Outcomes

  1. Explain the principles, benefits, and paradigms of Infrastructure as Code
  2. Write and manage Terraform configurations for cloud provisioning
  3. Manage state, modules, and variables in Terraform projects
  4. Create and manage Ansible playbooks for configuration management
  5. Implement infrastructure deployments on AWS, Azure, and Google Cloud
  6. Apply testing and validation strategies to infrastructure code
  7. Integrate infrastructure code into CI/CD pipelines
  8. Detect and remediate configuration drift in managed infrastructure

Course Outline

1 Infrastructure as Code Principles and Concepts 4 hours
Lesson: What is Infrastructure as Code?
  • IaC benefits and concepts
  • Declarative vs. imperative
Lesson: IaC Tools and Paradigms
  • Tool categories and selection
  • Terraform, Ansible, CloudFormation, Pulumi
Lesson: Best Practices
  • Code organization
  • GitOps principles

Lesson: Terraform Syntax and Providers
  • HCL structure
  • Resource definition
Lesson: State Management
  • State concept and backends
  • Remote state handling
Lesson: Modules and Reusability
  • Creating modules
  • Module composition
Lesson: Variables and Outputs
  • Input validation
  • Environment management
3 Configuration Management with Ansible 4 hours
Lesson: Ansible Fundamentals
  • Playbooks and YAML
  • Inventory management
Lesson: Modules and Tasks

  • Roles and organization

Lesson: AWS CloudFormation
