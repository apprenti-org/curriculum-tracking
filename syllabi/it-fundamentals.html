Syllabus: IT Fundamentals

IT Support Professional 80 hours DRAFT

Course Description

IT Fundamentals is a comprehensive 80-hour course providing foundational knowledge and practical skills for entry-level IT professionals. The curriculum covers essential topics including computer hardware and architecture, operating systems and software applications, networking fundamentals, cybersecurity basics, troubleshooting methodologies, data storage, and IT ethics. Learners develop both conceptual understanding and hands-on competency across the full IT technology stack.

Summative Assessment: Practical and Knowledge Exam

The IT Fundamentals assessment combines a hands-on practical exam (50%) and written knowledge exam (50%). The practical component requires learners to set up a Windows workstation, configure networking, implement basic security measures, and troubleshoot system issues. The knowledge exam includes multiple-choice and scenario-based questions covering all course modules. A minimum passing score of 80% is required.

Learning Outcomes

  1. Set up a computer workstation running Windows and use basic software applications
  2. Understand the functions and types of devices used within a computer system
  3. Apply basic computer maintenance and support principles
  4. Understand principles of software and database development
  5. Configure computers and mobile devices to connect to home networks and the internet
  6. Identify security issues affecting the use of computers and networks
  7. Explain computer system architecture, components, and performance factors
  8. Troubleshoot common hardware failures using diagnostic tools
  9. Install, configure, and optimize Windows operating systems
  10. Manage software applications and perform system maintenance
  11. Understand networking concepts, topologies, and protocols
  12. Configure IP addressing and network connections
  13. Apply basic network security and troubleshooting techniques
  14. Identify and respond to cybersecurity threats
  15. Implement basic security measures including firewalls and antivirus
  16. Apply systematic troubleshooting methodology to resolve IT issues
  17. Understand data storage options and backup strategies
  18. Implement disaster recovery and business continuity planning
  19. Understand IT ethics, data privacy laws, and professional conduct

Course Outline

1 Introduction to IT & Computer Hardware 7 hours
Lesson: Overview of IT and Industry Relevance
  • Introduction to Information Technology
  • IT career paths and industry demand
  • Computing device types and components
Lesson: Identifying Computer Components
  • Internal components: CPU, RAM, storage, motherboard, PSU, cooling
  • External components: input, output, peripheral devices
  • Hardware compatibility and performance
Lesson: Understanding System Architecture
  • Computer system architecture and data flow
  • BIOS/UEFI and boot process
  • System performance factors
Lesson: Basic Hardware Troubleshooting
  • Common hardware failures and symptoms
  • Troubleshooting tools and techniques
  • ESD precautions and best practices
2 Operating Systems & Software Applications 10 hours
Lesson: Introduction to Operating Systems
  • What is an operating system
  • Types of operating systems: Windows, macOS, Linux, mobile
  • Open-source vs proprietary systems
Lesson: Installing & Configuring Windows OS
  • OS installation preparation
  • Windows installation and partitioning
  • Post-installation configuration
Lesson: Working with Software Applications
  • Types of software: system vs application
  • Software installation and management
  • File management and productivity tools
Lesson: System Maintenance & Optimization
  • OS updates and patch management
  • Performance optimization techniques
  • Basic security best practices
3 Networking Fundamentals 8 hours
Lesson: Introduction to Networking Concepts
  • What is a network: LAN vs WAN
  • Network topologies: bus, star, mesh, ring, hybrid
  • Networking standards and protocols: TCP/IP, Ethernet, Wi-Fi, DNS, DHCP
Lesson: IP Addressing & Network Configuration
  • IPv4 vs IPv6 addressing
  • Subnetting and CIDR notation
  • Configuring network connections
Lesson: Basic Network Security & Troubleshooting
  • Common network security threats
  • Basic troubleshooting techniques: ping, tracert, ipconfig
  • Maintaining secure and stable networks
4 Security & Cybersecurity Basics 8 hours
Lesson: Introduction to Cybersecurity
  • What is cybersecurity
  • Common threats: malware, phishing, social engineering
  • CIA Triad: Confidentiality, Integrity, Availability
Lesson: Implementing Basic Security Measures
  • User account security and MFA
  • Configuring firewalls and antivirus
  • Safe browsing and email security
Lesson: Cybersecurity Best Practices & Incident Response
  • Recognizing and responding to security breaches
  • Introduction to encryption and secure data storage
  • Maintaining a secure IT environment
5 Troubleshooting & IT Support 8 hours
Lesson: Introduction to IT Troubleshooting
  • What is troubleshooting and the IT support role
  • Troubleshooting methodologies: 5-step process
  • IT support roles and responsibilities
Lesson: Diagnosing & Resolving Hardware and Software Issues
  • Common hardware issues and fixes
  • Software-related issues and compatibility
  • Using diagnostic tools and utilities
Lesson: Network & Security Troubleshooting
  • Common network issues and fixes
  • Diagnosing security issues
  • Best practices for IT support and documentation
6 Data Storage & Cloud Computing 8 hours
Lesson: Introduction to Data Storage
  • Types of storage: local vs cloud
  • Storage devices: HDD, SSD, NVMe
  • File systems: FAT32, NTFS, exFAT
Lesson: Cloud Storage & File Management
  • Cloud storage providers
  • Syncing and sharing files securely
  • Cloud storage security considerations
Lesson: Data Backup & Recovery
  • Backup strategies: full, incremental, differential
  • Backup and recovery methods
  • Disaster recovery planning
7 IT Ethics & Professionalism 5 hours
Lesson: IT Ethics & Legal Considerations
  • Understanding IT ethics
  • Data privacy laws: GDPR, HIPAA, CCPA
  • Intellectual property and software licensing
Lesson: Professionalism & Workplace Communication
  • Workplace expectations and professional conduct
  • Effective communication in IT support
  • Handling sensitive information

  • Network setup and configuration exercises
