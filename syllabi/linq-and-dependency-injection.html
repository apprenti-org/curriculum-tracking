Syllabus: LINQ and Dependency Injection

C# / .NET Fundamentals 40 hours DRAFT

Course Description

LINQ and Dependency Injection teaches advanced C# programming concepts essential for enterprise application development. Students will master exception handling, extension methods, LINQ using both method and query syntax, and dependency injection with Ninject. The course emphasizes functional programming approaches, error handling patterns, and architectural patterns for building loosely-coupled, testable applications. Students apply these concepts through progressive exercises building on the Weather Almanac application from prior modules.

Summative Assessment: Advanced C# Application with DI

Students will enhance the Weather Almanac application with comprehensive exception handling, LINQ-based reporting, and dependency injection architecture. The assessment requires proper exception handling patterns, both LINQ method and query syntax usage, proper DI configuration, and well-structured applications demonstrating architectural awareness. Learners must demonstrate understanding of inversion of control principles.

Learning Outcomes

  1. Implement exception handling using try/catch/finally blocks
  2. Create and use custom exceptions
  3. Apply error handling best practices
  4. Create and use extension methods
  5. Query data using LINQ method syntax
  6. Query data using LINQ query syntax
  7. Translate between method and query syntax
  8. Perform complex operations: filtering, selecting, ordering, grouping, aggregating
  9. Use lambda expressions effectively in LINQ
  10. Understand Inversion of Control (IoC) principles
  11. Configure and use Ninject dependency injection
  12. Implement constructor injection patterns
