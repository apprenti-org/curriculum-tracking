Syllabus: Advanced Python

IT Business Analyst DRAFT

Course Description

This Advanced Python course builds on foundational Python knowledge — variables, loops, conditionals, and functions — to introduce object-oriented programming concepts and practical web-based applications. Students explore the principle that "everything is an object" through lambda expressions, classes, magic methods, decorators, generators, and data classes. The course culminates with applying Python to real-world tasks including web scraping and working with APIs.

Students will gain hands-on experience designing and building Python classes, extending functionality with decorators and magic methods, and consuming data from web services. This course prepares students for roles that require intermediate-to-advanced Python skills and serves as a bridge between basic Python programming and professional software development.

Summative Assessment: Holiday Tracker (TBD Hours)

The Advanced Python Summative Assessment requires students to design and build a text-based Holiday Tracker application. Students must create Holiday and HolidayList classes using object-oriented programming concepts, load holidays from a JSON file and via web scraping (timeanddate.com), and provide a user interface for managing holidays (add, remove, save, view by week). The assessment integrates all course concepts: classes, magic methods, data classes, web scraping, and API interaction.

Students are assessed using a competency-based rubric that measures proficiency in OOP design, code structure, and practical application of advanced Python concepts. A minimum passing score of 80% is required to demonstrate readiness for real-world Python development tasks.

- Design a text-based Holiday Tracker application

- Load holidays from JSON file and web scraping (timeanddate.com)

- Create Holiday and HolidayList classes using OOP concepts

- Apply magic methods, data classes, and API integration

- User interface for managing holidays (add, remove, save, view by week)

- Starter files: holiday_startercode.py, holidays.json

Learning Outcomes

  1. Identify the purpose of lambda expressions in Python.
  2. Use `filter()`, `map()`, and `reduce()` to manipulate data.
  3. Identify the four pillars of object-oriented programming.
  4. Create classes with properties, methods, getters, and setters.
  5. Apply inheritance, composition, and interfaces in Python.
  6. Define and use magic methods (dunder methods) for object lifecycle and comparisons.
  7. Create and apply custom decorators to extend functions and classes.
  8. Differentiate between iterators and generators and create both.
  9. Describe the purpose of a data class and create data classes in Python.
  10. Identify ethical and legal concerns of web scraping.
  11. Consume REST APIs using the `requests` library.
  12. Apply all course concepts to build a comprehensive Python application.
