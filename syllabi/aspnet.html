Syllabus: ASP.NET

C# / .NET Fundamentals DRAFT

Course Description

ASP.NET Core Fundamentals introduces learners to web development using the ASP.NET Core framework. Students will explore the HTTP protocol, the Model-View-Controller (MVC) architectural pattern, and the development of RESTful Web APIs. The course covers model validation, secure authentication using JWT tokens, and practical implementation patterns for building modern web applications. This course prepares developers for full-stack web development roles and deepens their understanding of server-side architecture in the .NET ecosystem.

Summative Assessment: ASP.NET Core Application Project

Students will build a complete ASP.NET Core application that integrates multiple module concepts. The assessment requires the implementation of both MVC views and Web API endpoints, application of model validation, and secured API endpoints using JWT authentication. This capstone project demonstrates full-stack C# web development proficiency and understanding of RESTful principles, authentication patterns, and the architectural separation between presentation and service layers.

Learning Outcomes

  1. Understand the HTTP protocol, including request/response models and status codes
  2. Explain the Model-View-Controller (MVC) architectural pattern
  3. Create ASP.NET Core projects and implement controllers, views, and models
  4. Implement CRUD operations using the MVC pattern
  5. Build RESTful Web APIs using ASP.NET Core
  6. Apply model validation using data annotations and custom validators
  7. Implement authentication and authorization using JWT tokens
  8. Configure middleware and protect API endpoints with the [Authorize] attribute
  9. Test APIs using Swagger and Postman
