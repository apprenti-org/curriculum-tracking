Syllabus: Software Developer Pre-Work

Software Development Fundamentals 20 hours DRAFT

Course Description

Software Developer Pre-Work is an introductory course designed to prepare incoming software development apprentices with essential computer fundamentals and workplace readiness. Using a relatable kitchen metaphor, students will explore how computers work, operating systems, file systems, internet usage, networking, and software management. This course establishes the foundational knowledge needed for all software development programs and ensures learners are comfortable with basic computer operations before progressing to programming-focused courses.

Summative Assessment: Computer Fundamentals and Troubleshooting

Students will complete a final assessment evaluating their understanding of computer components, operating systems, file systems, networking, and software installation. The assessment includes both theoretical questions and practical troubleshooting scenarios. Learners must demonstrate systematic problem-solving approaches and understand how computer systems work together.

Learning Outcomes

  1. Explain hardware components and their functions (CPU, RAM, storage, peripherals)
  2. Understand the role of operating systems in computer functionality
  3. Navigate file systems effectively across different operating systems
  4. Understand networking fundamentals and connectivity
  5. Use the internet safely and responsibly
  6. Install and manage software appropriately
  7. Troubleshoot common computer and connectivity issues
  8. Understand basic network concepts (IP addresses, DNS, protocols)
  9. Apply systems thinking to problem-solving
  10. Use command-line basics (introduction)

Course Outline

1 Basic Computer & OS Navigation 4 hours
Lesson: 1.1: What is a Computer? (2 hours)
  • Hardware vs software
  • CPU, RAM, storage (HDD/SSD), input/output devices
  • Data lifecycle: input → process → output
  • Computer architecture overview
Lesson: 1.2: Understanding Operating Systems (2 hours)
  • What is an OS? (Windows, macOS, Linux)
  • Graphical User Interface (GUI) fundamentals
  • Exploring system settings
  • Intro to Command Line (optional)
2 File Systems & Internet Use 4 hours
Lesson: 2.1: Navigating File Systems (2 hours)
  • Folder structure and hierarchy
  • File types and extensions (.docx, .jpg, .exe, etc.)
  • File properties and permissions
  • Basic file operations: move, rename, delete, search
Lesson: 2.2: Using the Internet Safely (2 hours)
  • How browsers work and URLs
  • Bookmarks, tabs, and browser history
  • Safe browsing practices
  • Phishing awareness and security
  • Downloading files responsibly
3 Networking Fundamentals 4 hours
Lesson: 3.1: What is a Network? (2 hours)
  • LAN, WAN, Wi-Fi basics
  • IP addresses and how they work
  • DNS and domain names
  • How data travels (packets, protocols)
Lesson: 3.2: Network Devices & Issues (2 hours)
  • Routers, switches, and modems
  • Common connectivity issues
  • Troubleshooting network problems
  • Network performance concepts
4 Installing & Managing Software 4 hours
Lesson: 4.1: Installing Software Safely (2 hours)
  • Trusted sources and repositories
  • Installation processes and wizards
  • Permissions and admin access
  • Avoiding malware and unwanted software
Lesson: 4.2: Managing Software & Antivirus (2 hours)
  • Updates and patch management
  • Uninstalling software cleanly
  • Antivirus and security tools
  • System maintenance
5 Practice, Troubleshooting & Final Check 4 hours
Lesson: 5.1: Troubleshooting Scenarios (2 hours)
  • Real-world troubleshooting exercises
  • Systematic problem-solving approach
  • When to seek help
Lesson: 5.2: Final Challenge & Quiz (2 hours)
  • Summative assessment
  • Skills verification
  • Readiness check for software development

Labs & Practical Exercises

  • Hardware Exploration: Open a computer case, identify components (simulator or hands-on)
  • Operating System Navigation: Work with Windows, macOS, or Linux
  • File System Practice: Create folders, move files, understand hierarchy
  • Network Troubleshooting: Diagnose and resolve connectivity issues
  • Software Installation: Install and uninstall applications safely
  • Internet Safety: Recognize phishing and unsafe practices
  • Problem-Solving: Work through troubleshooting scenarios
