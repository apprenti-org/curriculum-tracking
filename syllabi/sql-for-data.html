Syllabus: Data Fundamentals — SQL for Data

Data Analyst (DANL) 50 hours DRAFT

Course Description

SQL for Data teaches database querying and data analysis using SQL. Students will master data analysis fundamentals, Docker containerization, SQL querying from basic SELECT statements to advanced joins and subqueries, data definition and manipulation, and business applications of SQL. The course uses practical datasets and tools (Docker, SQL Server, Azure Data Studio) to prepare data professionals for real-world analytics work. This course is foundational for all advanced data analysis paths.

Summative Assessment: Data Analysis and SQL Reporting Project

Students will complete a comprehensive assessment analyzing real-world datasets using SQL. The assessment requires writing complex queries with joins and subqueries, performing aggregations and grouping, creating data definitions, and translating business questions into SQL. Learners must demonstrate proficiency with database design, query optimization, and business-focused reporting.

Learning Outcomes

  1. Understand data analysis fundamentals and data-driven decision making
  2. Model data and normalize databases
  3. Set up and navigate SQL Server and development environments
  4. Write SELECT queries with filtering and ordering
  5. Perform aggregations and grouping operations
  6. Work with date functions and calculations
  7. Join multiple tables to combine data
  8. Write subqueries for nested queries
  9. Create and modify database objects (DDL)
  10. Manipulate data (DML)
  11. Apply SQL to business scenarios and reporting
  12. Optimize queries and understand performance
